# 应用部署手册

## 概述

本文档记录了应用的部署流程和相关配置说明。

---

## 前置要求

在开始部署前，先进行以下工具的安装：

- **Node.js** >= v22.0.0
- **pnpm**
- **Git**
- 注册好 AI 大模型开放平台的账号，并成功获取到 **API Key**（本手册以 DeepSeek 为例）

---



## 安装步骤

### 1. 安装 Node.js

1. 访问 Node.js 官网：[Node.js — Download Node.js®](https://nodejs.org/)
2. 点击**Get Node.js**绿色按钮
3. 选择带有 **LTS** 标识的版本
4. 点击 **Windows Installer (.msi)** 下载安装包
5. 运行安装程序并按照提示完成安装


### 2. 安装 pnpm

1. 确认 Node.js 已成功安装
2. 打开 **CMD** 命令行终端
3. 输入以下命令安装 pnpm：

```bash
npm install -g pnpm
```


### 3. 安装 Git

在 Windows 上安装 Git 有以下几种方法：

#### 方法一：官方安装包

1. 访问 Git 官方网站：https://git-scm.com/download/win
2. 下载会自动开始
3. 运行安装程序并按照提示完成安装

> 注意：这是一个名为 Git for Windows 的项目（也叫做 msysGit），和 Git 是分别独立的项目。更多信息请访问 http://msysgit.github.io/

#### 方法二：使用 Chocolatey 包管理器

使用 Git Chocolatey 包进行自动安装。

> 注意：Chocolatey 包是由社区维护的。

#### 方法三：安装 GitHub Desktop

安装 [GitHub Desktop](https://desktop.github.com/)，该安装程序包含：
- 图形化版本的 Git
- 命令行版本的 Git
- 支持 PowerShell
- 稳定的凭证缓存
- 健全的换行设置

> 参考来源：[Git 官方文档](https://git-scm.com/book/zh/v2)

## 部署方式

### 方式一：使用 npm 直接安装（推荐）

这是最简单的安装方式，只需一条命令：

```bash
npm install -g openclaw@latest
```

### 方式二：克隆仓库部署

先克隆项目仓库到本地：


> **提示**：如果使用克隆仓库部署，接下来相关命令如果出现无法识别的情况，在前面加上 `pnpm` 即可解决。

#### 步骤 1：克隆仓库

1. 在想要保存项目的路径打开命令行
2. 输入以下命令克隆仓库：

```bash
git clone https://gitee.com/OpenClaw-CN/openclaw-cn.git
cd openclaw-cn
```

完成后，仓库将被克隆到本地。


#### 步骤 2：构建项目

依次执行以下命令进行项目构建：

```bash
# 安装依赖
pnpm install

# 构建前端界面，首次运行时自动安装 UI 依赖
pnpm ui:build

# 构建核心服务
pnpm build
```

> **故障排除**：如果在这一步出现报错，排除网络问题后一般是库不完整的问题，需要安装 Visual Studio Build Tools（必须）：

「Desktop development with C++」（必选）

勾选「Windows 10/11 SDK」（根据你的系统选择）
点击「安装」（约占用 5-10GB 磁盘空间）


#### 步骤 3：初始化 OpenClaw

构建完成后，在相同路径运行以下命令进行初始化：

```bash
pnpm openclaw onboard --install-daemon
```



#### 步骤 4：配置 AI 模型

> **提示**：本手册以 DeepSeek 和飞书为例进行配置说明。

初始化脚本启动后，按照提示进行配置：

1. 选择 **DeepSeek** 模型
2. 输入从 [DeepSeek 开放平台](https://platform.deepseek.com/) 获取的 API Key：

```
sk-xxxxxxxxxxxxxx
```

## 启动服务

初始化完成后，网关会自动启动。若没有自动启动，则在相同路径输入以下命令：

```bash
# 启动网关 (Gateway)
pnpm openclaw gateway
```


---

## 配置飞书事件回调

确定网关启动后，切换回飞书开放平台进行以下配置：

### 1. 配置事件

1. 左侧菜单进入 **事件与回调** > **事件配置**
2. 请求方式选择：**使用长连接接收事件**
   
   > **重要**：这是关键配置，使用长连接方式不需要公网服务器！

3. 添加事件：
   - 搜索 `im.message.receive_v1`（接收消息）
   - 勾选并添加
   
   > **重要**：必须添加此事件，否则接收不到消息。

### 2. 发布应用

1. 左侧菜单进入 **版本管理与发布**
2. 点击 **创建版本**
3. 填写必填项
4. 提交发布


---

## 配对机器人

最后一步，完成机器人与用户的配对：

1. 登录飞书 App
2. 进入 **更多** > **工作台** > **最近使用**
3. 找到刚才创建的机器人
4. 随便发送一条消息
5. 机器人会回复配对码
6. 复制配对码，在终端运行以下命令：

```bash
openclaw pairing approve feishu <配对码>
```

配对成功后，即可正常使用机器人功能。


---

## 开机自启（可选）

以上就是配置的全部流程。如需设置机器人开机自启，可运行以下命令：

```bash
openclaw gateway install
```
